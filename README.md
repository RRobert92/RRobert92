  ### 🚀 Featured projects                                                                                                            

  **[TARDIS-EM](https://github.com/SMLC-NYSBC/TARDIS)** — *Transformer And Rapid Dimensionless Instance Segmentation*<br/>
  Deep-learning framework for fast, accurate instance segmentation of filaments (microtubules) and membranes in 2D/3D (cryo-)EM micrographs
  and tomograms — a CNN semantic step paired with **DIST**, a point-cloud segmentation transformer.<br/>
  ↳ `pip install tardis-em`
    
  - [TARDIS_EM](https://github.com/SMLC-NYSBC/TARDIS) · [napari plugin](https://github.com/RRobert92/napari-tardis_em) · [tardis-em-analysis](https://github.com/RRobert92/tardis_em_analysis) · [paper](https://www.biorxiv.org/content/10.1101/2024.12.19.629196v1)

  **[PANDORICA](https://github.com/RRobert92/pandorica)** &nbsp;🚧 _work in progress_<br/>
  A tool for the EM community enabling a **fully automated, high-throughput** pipeline — from data pre-processing all the way to final
  results analysis. Stitches imperfectly matched, locally deformed serial sections into one coherent 3D volume (coarse global alignment,
  then a diffeomorphism-guarded thin-plate-spline warp) in minutes on an ordinary desktop — no server or GPU required.<br/>
  ↳ `pip install pandorica`
  
  - [follow along on GitHub](https://github.com/RRobert92/pandorica) - in active development

  **[EM ML Tools Overview](https://github.com/RRobert92/EM_ML_Segmentation_Tools_Overview)**<br/>
  A searchable guide that helps readers find the right ML tool for an EM task.

  **[ASGA](https://github.com/RRobert92/ASGA)**<br/>
  Open-source spindle / microtubule spatial-graph analysis for ZIB Amira output.
